ngTabs
A powerful responsive tabs widget

Usage

ngTabs is the easiest way to put a simple and responsive tabs widget in your website.
When there are more items then visible, arrows will be shown and let you navigate!
  1. Import JS and CSS in your page
  2. Inject ngTabs in your app: 
    var app = angular.module("myApp", ["ngTabs"]);
  3. Put this simple snippet where you want the widget to appear: 
    
                            <ng-tabs>
                            <ng-tabs-pane ng-tabs-title="TAB 1 TITLE">
                            </ng-tabs-pane>
                            </ng-tabs>

Options

Option Description Type Default value
visible-items-desktop Number of elements to display on desktop (width > 992px) Number 3
visible-items-tablet Number of elements to display on tablet (768px <= width >= 992px) Number 3
visible-items-tablet Number of elements to display on mobile (width < 768px) Number 1
show-indicators If "true" one dot per tab will be showed over the tabs. String false
force-visible By default, ng-tabs will control if visible items is higher than the number of tabs. if force-visible is "true", the control will be skipped and you could achieve to display tabs as they are in the second demo below. Boolean true
ng-tabs-title This param is to put on the <ng-tabs-pane> element. The title will be put in the tab toggle string en
